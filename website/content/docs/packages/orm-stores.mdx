import { Callout } from 'fumadocs-ui/components/callout';

The `media` table is persisted by a `MediaStore`. Every adapter is a **POJO that receives your ORM handle** in its constructor (no `@Injectable`, no internal token) and satisfies the same conformance suite. Pass one to `MediaModule`'s `store` option.

## TypeORM — `@dudousxd/nestjs-media-database-typeorm`

```ts
import { TypeOrmMediaStore, ensureMediaSchema } from '@dudousxd/nestjs-media-database-typeorm';

new TypeOrmMediaStore(dataSource);                       // autoCreateSchema: true (default)
new TypeOrmMediaStore(dataSource, { autoCreateSchema: false }); // you manage migrations
await ensureMediaSchema(dataSource);                     // or run it yourself
```

Auto-schema is **non-destructive**: creates the table and adds missing columns, never drops or alters. Timestamps are stored as portable ISO strings so the schema works on sqlite, mysql, and postgres alike.

## MikroORM — `@dudousxd/nestjs-media-database-mikro-orm`

```ts
import { MikroOrmMediaStore, ensureMediaSchema } from '@dudousxd/nestjs-media-database-mikro-orm';

await ensureMediaSchema(orm);            // orm.schema.updateSchema({ safe: true })
new MikroOrmMediaStore(orm.em);
```

## Drizzle — `@dudousxd/nestjs-media-database-drizzle`

```ts
import { DrizzleMediaStore, createMediaTable, mediaTable } from '@dudousxd/nestjs-media-database-drizzle';

createMediaTable(db);                     // dev/test helper; production uses drizzle-kit
new DrizzleMediaStore(db);
```

<Callout type="info" title="Drizzle is sqlite-first">
Drizzle's schema is dialect-specific (`sqliteTable` vs `pgTable`), so this adapter targets sqlite/libSQL and is migration-first — matching the ecosystem's other Drizzle adapters. Bring your own schema for other dialects.
</Callout>

## Prisma — `@dudousxd/nestjs-media-database-prisma`

Schema is consumer-managed; the adapter never imports `@prisma/client`. Add a `Media` model to your schema (map `order` → a `position` column), then:

```ts
import { PrismaMediaStore } from '@dudousxd/nestjs-media-database-prisma';

new PrismaMediaStore(prisma); // your PrismaClient
```

The structural `PrismaClientLike` contract is validated against a real generated client + Postgres in the package's integration suite.
